## ✨ Features

| MVP Core | ✅ Working |
|----------|-----------|
| 👤 **Auth** | Email + Google OAuth, JWT cookies |
| 📁 **Folders** | Nested hierarchy, breadcrumbs, drag-drop |
| 📎 **Files** | Drag-drop upload (resumable), progress, thumbnails |
| 🔗 **Sharing** | User invites (Viewer/Editor), public links (expiry + password) |
| 🔍 **Search** | Name/type/owner filters, full-text, fuzzy matching |
| ⭐ **Quick Access** | Recent, Starred, Trash (30-day retention) |
